### 创建项目
```
npm init
npm i express

npm i 就是根据package.json以及package-lock.json下载包


npm uninstall express

npm i express@3 //安装指定版本， 3的最新版本
npm i express@latest


npm outdated 列出目前的版本以及新的版本
npm update
npm i express@latest

```


```
npm i -D nodemon // 下载nodemon到Devdependency

npx 临时全局安装
npm i -g nodemon 全局安装，不推荐全局安装
```
